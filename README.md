# DR_classification
The aim of this project is to classify the input eye image into following 5 classes:
  1) Mild
  2) Moderate
  3) No Dr
  4) PDR
  5) Severe

Features Extractors
  1) Resnet152
  2) EfficientB0
  3) VGG16

Feature Selection Techniques
  1) PCA
  2) Feature Importance

Classifier 
  1) Random Forest
  2) ANN
